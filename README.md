# README

## Dependencies

* Ruby >= 2.3.1
* Rails >= 5
* PostgreSQL >= 9.5

## Usage 
```
gem install bundle
bundler install
rails s
```

## Deployment
```
heroku login 
git remote add heroku https://git.heroku.com/Zalogika.git
git push heroku master
```
